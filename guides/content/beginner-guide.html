Introduction

Bienvenue dans ce guide de démarrage pour TopGirl ! Ce guide voustera à travers les mécaniques de base du jeu pour vous aider à commencer votre aventure du bon pied.

TopGirl est un jeu de collection et de stratégie où vous devrez rassembler des artistes, améliorer vos personnages et combattre pour devenir le meilleur.

L'interface du jeu

L'interface de TopGirl peut sembler complexe au début, mais elle est très intuitive une fois comprise. Voici les éléments principaux :

  • En haut à gauche : Vos ressources (Or, Gemmes, Energie)
  • En haut à droite : Votre profil et paramètres
  • Au centre : Le contenu principal du jeu
  • En bas : La navigation principale (Accueil, Personnagers, Combat, Boutique)

Vos premiers pas

Voici ce que vous devez faire dès le début :

  1. Complétez le tutoriel : Il vous guidera à travers les bases
  2. Rassemblez vos premiers artistes : Utilisez les pulls gratuits
  3. Constituez votre équipe : Mixez différents types pour créer des synergies
  4. Faites vos premiers combats : Apprenez le système de combat

Gestion des ressources

Les ressources sont essentielles dans TopGirl. Voici les principales :

  • Or : La devise principale, utilisée pour presque tout
  • Gemmes : La premium currency, pour les pulls et avantages
  • Énergie : Nécessaire pour jouer, se régénère avec le temps
  • Matériaux : Pour upgrade et évoluer vos personnages

Conseils pour débuter

  • Ne gaspillez pas vos ressources : Especially au début, savez vos gemmes pour les événements
  • Concentrez-vous sur une équipe : Il vaut mieux avoir 5 personnages forts que 10 faibles
  • Faites les défis quotidiens : Ils donnent des récompenses importantes
  • Rejoignez la communauté : Le Discord et les forums regorgent de conseils
  • jouez régulièrement : Les login rewards sont très généreuses
